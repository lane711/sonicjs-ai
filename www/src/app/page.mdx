## Quick Start in 60 Seconds {{ id: 'quick-start' }}

Get SonicJS running locally in under a minute:

```bash {{ title: 'Installation' }}
# Clone the repository
git clone https://github.com/lane711/sonicjs.git
cd sonicjs

# Install dependencies
npm install

# Run migrations and start dev server
npm run db:migrate
npm run dev

# Your CMS is now running at http://localhost:8787
```

Visit the [Quickstart guide](/quickstart) for detailed instructions and next steps.
